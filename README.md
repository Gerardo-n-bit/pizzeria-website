# Pizzeria Da Jerry - Mini Sito

Questo è un piccolo progetto personale che mostra le basi di HTML, CSS e JavaScript.  
Il sito permette di:

- Visualizzare il menu delle pizze e selezionare quantità  
- Calcolare il totale dell'ordine  
- Confermare o annullare un ordine  
- Prenotare un tavolo e aggiornare il numero di tavoli disponibili

> **Nota:** questo progetto è un esercizio di base per dimostrare competenze front-end minime. Il mio focus principale è la **programmazione a oggetti** e lo sviluppo back-end.

## Tecnologie utilizzate

- HTML per la struttura della pagina  
- CSS per lo stile e il layout responsive  
- JavaScript per l’interattività e la logica dei calcoli

## Come usare

1. Apri il sito live o scarica il repository  
2. Seleziona una pizza e inserisci la quantità, poi clicca su "Calcola Totale"  
3. Conferma o annulla l’ordine  
4. Prenota un tavolo indicando il numero di posti e il cognome
